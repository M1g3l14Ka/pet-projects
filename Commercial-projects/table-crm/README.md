## Features
  A commercial product creation form built with Next.js 15 and Server Actions. Features secure API integration and a custom Gemini AI module that automatically generates SEO metadata based on user input

## 🛠 Tools & Tech Stack
- Core: Next.js, TypeScript
- Styling: Tailwind CSS, shadcn/ui
- Integrations: Gemini API

## 🤓 What I Learned
- Deepened my knowledge of Tailwind CSS with grid layouts.
- Successfully integrated a gemini API. 
- Improved overall Next.js project architecture and component isolation.

## 🤔 Future Improvements
- Add smoother, highly complex animations.
- Adding more micro-interactions for better user engagement.

# 🛜 How to Run Locally
If you want to run this project on your machine:
1.	Download the archive with the project
2.	Open the project in an editor that is convenient for you (I use Vs code)
3.	Important:
- Create a .env.local file in the root directory and add your Resend API key to make the contact form work: GEMINI_API_KEY=your_api_key_here
5.	Open a terminal and type npm run dev
6.	You will launch the project at http://localhost:3000/
